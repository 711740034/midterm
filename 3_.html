新聞隨堂考

觀念釐清題：以下敘述是否正確？

Question 1

假資訊主要傳播管道是大眾媒體。

Feedback

False

社群軟體、口耳相傳都可以是假資訊傳播的管道。

Question 2

假新聞是這時代獨有的現象。

Feedback

False

不管在哪個時代都有假新聞存在，只是傳播管道的不同。

Question 3

假資訊無法透過個人行動削弱它的傳播力道。

Feedback

False

人人都可以從「不轉傳未經證實的消息」做起！
